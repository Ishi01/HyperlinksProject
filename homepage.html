Introduction

Hyperlinks are one of the core concepts that allow the internet as we know it today to function. In the simplest terms, the function of a hyperlink is to redirect the user from where they currently are on a webpage to somewhere else, either somewhere in the same page, a different page on the same website, a different website entirely, or pretty anywhere you can go on the internet. The hyperlink can also direct to a specific section of the same, or another, webpage using an anchor tag. It can also initiate the download for an external resource or file, or provide an expanded view of an image or video.


Implementation of Hyperlinks

From the web-developers perspective, the implementation of hyperlinks varies depending on the purpose of the link. For on-page links, the implementation relies primarily on the use of an anchor tag. These tags allow you specify an href attribute, which is usually the ID of a page element which you want the user to jump to when the anchor tag is clicked. Anchor tags can be attached to virtually any element in a web page, but are most commonly attached to either buttons or links, using the classic blue underlined text, which has become the ubiquitous UI language for pretty much any form of link.
